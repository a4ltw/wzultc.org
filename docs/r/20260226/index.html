多語照護新教材開拍

Quay phim tài liệu chăm sóc đa ngữ

掃描分享本頁

文藻外語大學團隊與高雄榮民總醫院、高雄醫學大學及輔英科技大學攜手合作，打造給長照場域越南籍照服員專屬的專業技能示範影片！

活動由廖俊芳主持人帶領，說明拍攝目標與教材用途。高醫的陳建州老師蒞臨現場指導，提醒示範教材要兼顧正確性與細節呈現，才能讓影片成為照服員自學時的實用工具。

榮總的淑芬阿長與桂珍組長不僅全程參與，更親自上場協助拍攝，從專業角度提供許多寶貴建議，讓專業示範動作更貼近實際照護現場。趙立記者親自協助拍攝與指導取景，讓整個拍攝流程順利又充滿跨域交流學習的好氣氛。

輔英科技大學的越南籍實習生李珠芳儀、阮草原也加入演出及協助工作，透過親身參與，深入體會長照照護工作的要點與精神。

團隊將持續製作多語專業技能照護教材，期盼在培訓、增能外籍照服員時，提供實務、易於理解的自學素材，幫助他們更快融入適應台灣文化與長照職場環境。

敬請期待成果影片上線！
