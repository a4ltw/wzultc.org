# 歡迎來到 WZULTC 越南學伴計畫

## 關於我們

WZULTC（越南學伴計畫）致力於為新移工提供溫暖的陪伴與支持，透過會越南語與華語的學伴，協助他們適應台灣生活，同時促進跨文化交流與理解。

## 🌟 我們的願景

- **傳遞善意**：為遠道而來的朋友提供溫暖支持
- **文化交流**：促進台越文化互相理解與尊重
- **共同成長**：讓參與者在服務中獲得跨領域經驗
