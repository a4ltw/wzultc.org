✨ 想成為越南學伴嗎？ ✨

會越南語＋華語的你，發光的時刻到了！陪伴新移工適應生活，同時拿下實習學分、志工服務時數及跨文化經驗，一次全包，機會別錯過！✨

✅ 實習72小時（2學分）

✅ 志工30/60小時（認列境外生獎學金志工時數）

  • 線上陪伴：分享生活與語言學習經驗，給自己家鄉來的朋友溫暖支持

💡 為什麼要加入？因為：

  • 能成長｜跨領域、拓視野

  • 母語為越南語，會說華語
  • 樂於助人，願陪伴他人
  • 至少能完成30小時服務
